# DIW_Maquetacio

Repositori per l'activitat de l'assignatura de Disseny d'Interfícies Web
Els objectius són:
- Reconeixer les possibilitats de modificar les etiquetes HTML.
- Definir i associar estils globals a fulls externs.
- Redefinir estils.
- Identificar les diferents propietats de cada element.
- Crear classes d'estils.
- Utilitzar eines de validació de fulls d'estils.
